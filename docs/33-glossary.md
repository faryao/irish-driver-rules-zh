# 术语表

|英文术语|中文解释|
|---|---|
|Abreast|并排。|
|Acceleration|加速。|
|Arrhythmia|心律不规则或异常。|
|Axle|连接车辆左右相对车轮的一根轴杆。|
|Blind spot|驾驶人无法直接或通过后视镜看见的区域，须转头或侧望才能发现其他道路使用者。|
|Blow out|轮胎突然失效或爆裂。|
|Build outs|从道路一侧向车行道伸出的路缘，用于降低车速。|
|Cardiovascular diseases|涉及心脏和血液循环系统的疾病。|
|Central median island|道路中央分隔对向车流或供行人过街的区域。|
|Central nervous system|大脑和脊髓。|
|Cerebrovascular diseases|涉及脑部血管的疾病。|
|Certificate of Professional Competency (CPC)|以驾驶重型货车或巴士为职业前必须持有的职业能力证书。|
|Chicane|使车辆减速并在车道间曲折行驶的交通缓和设施。|
|Chevron board|带人字形斜线、表示行驶方向突然改变的警告标志。|
|Cross-ply tyres|帘线以不同角度交叉、用于加强胎侧和胎面的斜交轮胎。|
|Deceleration|减速。|
|Defects|缺陷，例如后视镜破损或车灯缺失。|
|Design Gross Vehicle Weight (DGVW)|制造商规定的设计车辆总重量。|
|Diverging|分开，例如一部分车流右转而其他车辆直行，或车辆驶离高速公路。|
|Fixed-wheel bicycle|可通过反踩制动、其中一个车轮不能独立于脚踏旋转的固定齿轮自行车。|
|Gantries|横跨车行道、用于悬挂标志的头顶钢结构。|
|Garda Síochána|爱尔兰国家警察。|
|Ghost island|高速公路与匝道交汇处的道路标线区域，又称导流区。|
|Graduating|从一个阶段进至下一阶段。|
|Gross Vehicle Weight (GVW)|车辆自重与其设计最大载荷之和。|
|Hard shoulder|以黄色虚线或实线与道路其余部分分隔、只供特定道路使用者在特定情况下使用的硬路肩。|
|Hatched marking|帮助分隔车道的斜线或人字形道路标线。|
|Hazard|道路上任何可能造成危险的事物。|
|Intoxicant|可影响行为、感知、情绪或警觉性的物质。|
|Invalid carriage|专为残障人士设计或制造的车辆；不包括经特殊改装、获准使用高速公路的普通汽车。|
|MAM|最大许可质量（Maximum Authorised Mass）。|
|National road|连接城市地区的主要道路，包括以 `M` 编号的高速公路和以 `N` 编号的其他路线。|
|Land tractor|通常称农业拖拉机，为农业、林业或土地排水作业而设计，只在往返作业地点时使用公共道路。|
|Lighting-up hours|从日落后半小时开始，到翌日日出前半小时结束的法定开灯时段。|
|Luas|都柏林市区及郊区的电车系统。|
|Manoeuvre|操纵或改变车辆路线的任何动作，例如起步、变道、驶离环形交叉路口、转弯、掉头或倒车。|
|Median space|双车道公路中央分隔带内供车辆横过或转入另一道路的空隙。|
|Merging|汇合，例如入口匝道车辆汇入高速公路车流。|
|Moped|发动机排量不超过 50 cc、最高速度 45 km/h 的轻型摩托车。|
|Motorcycle engine capacity|发动机排量，以立方厘米 `cc` 表示。|
|Motorcycle power rating|发动机输出功率。|
|Muscular atrophy|肌肉萎缩。|
|Nearside|左侧。|
|Negligence|未给予合理谨慎。|
|Night-driving mode (mirror)|降低眩光的后视镜暗色反射模式。|
|Non-national road|连接郡或地区内村镇、以 `R` 或 `L` 编号的地方或区域道路。|
|Offside|右侧。|
|Ophthalmic optician|有资格验配眼镜和隐形眼镜并发现眼病的验光师。|
|Outer lane|双车道公路或两、三车道高速公路上最靠近道路中央的车道。|
|Permit|允许停车或使用特定道路等行为的法律文件。|
|Pinch points|收窄部分道路以降低车速的交通缓和设施。|
|Pointsman|指挥车流的爱尔兰警察。|
|Prescription|医生或医院说明应服用何种药物及服用时间的书面处方。|
|Professional drivers|主要收入来自驾驶工作的人员，例如巴士、长途客车和货运驾驶人。|
|Radial tyres|帘线在胎面下方按子午方向排列、用于加强和延长寿命的子午线轮胎。|
|Retarder|不使用常规制动器而降低车辆速度的装置。|
|Safe headway|两辆车之间的安全距离。|
|Secondary controls|不直接控制车辆移动或制动，而控制驾驶人向外视野及车辆可见度的装置，例如除雾器、雨刷、清洗器和危险警告灯。|
|Single lane for shuttle working|道路施工区以人工停车／前进牌控制的单车道交替通行。|
|Stationary|处于停止状态，例如在车流、红灯或停车位中。|
|Swan neck|驾驶人越过正确右转点后，为修正位置而采用的过大弯曲路线。|
|Tabard|摩托车骑手穿着、前后清楚展示 `L` 标志的无袖黄色荧光背心。|
|Tachograph|测量并记录车辆速度、距离和行驶时间的设备。|
|Tailgating|过于贴近前车驾驶。|
|Tarpaulin|覆盖运输货物的防水布。|
|T-junction|次路与主路相交形成 `T` 形的交叉路口。|
|Tread|提供道路抓地力的轮胎沟槽。|
|Variable message sign|内容可变化、用于提供道路和安全信息的电子标志。|
|Vigilant|谨慎、警觉并留意可能的危险。|
|Work vehicles|通常不在普通道路行驶、用于工地或道路施工的车辆。|
|Yield|让其他道路使用者先行。|
