# 附录1：需要医疗报告的病症

患有下列疾病或残障时，申请驾驶执照必须提交医疗报告：

- 使用胰岛素或磺酰脲类药片治疗的糖尿病；不确定时询问医生。仅以其他药片或饮食控制者无需报告。
- 癫痫。
- 中风或短暂性脑缺血发作（TIA），并有任何相关症状持续超过一个月。
- 痉挛发作或昏厥。
- 任何脑部手术；需要住院治疗的脑脓肿或严重头部损伤；脑肿瘤、脊柱损伤或脊柱肿瘤。
- 植入式心脏起搏器。
- 植入式心律转复除颤器（ICD）。
- 反复突然发作、使人失去行动能力的眩晕。
- 其他慢性神经系统疾病，例如多发性硬化、运动神经元病、帕金森病或亨廷顿病。
- 严重记忆问题或间歇性意识混乱。
- 持续滥用或依赖酒精。
- 持续滥用或依赖药物。
- 严重精神疾病或心理健康问题。
- 帕金森病。
- 睡眠呼吸暂停综合征。
- 发作性嗜睡病。
- 影响周边视野的任何病症。
- 一只眼睛完全失明。
- 影响双眼的任何病症；如果只有一只眼睛，则包括影响剩余眼睛的病症，但不包括色盲、近视或远视。
- 自上次申请或续领后恶化的严重听力缺陷。
- 手臂或腿部持续问题，以致驾驶必须限于某些车辆类型或配有改装控制装置的车辆。
- 严重学习障碍。

本清单并非穷尽。医疗报告要求的更多信息请参阅 [RSA](https://www.rsa.ie) 或 [NDLS](https://www.ndls.ie)。
