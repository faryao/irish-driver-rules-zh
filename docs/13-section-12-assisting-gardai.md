# 第12节：配合爱尔兰警察

爱尔兰国家警察（An Garda Síochána）负责执行道路交通法。本节说明道路上必须遵从的警察信号和指示。

## 信号

爱尔兰警察指挥交通时，其信号优先于交通灯和其他所有信号。例如，即使交通灯为绿色，警察示意停车时也必须停车。

原书图示的警察信号包括：

- 示意前方接近的车辆前进；
- 示意两侧接近的车辆前进；
- 使前方接近的车辆停车；
- 使后方接近的车辆停车；
- 使前后两个方向接近的车辆都停车。

必须理解这些信号，以便在车流中正确应对。

## 指示

爱尔兰警察提出要求时，必须：

- 出示驾驶执照；驾驶时必须始终随身携带。
- 允许检查车辆展示的保险、车辆税及适用的 NCT 和 CVRT 标志。
- 在十天内到指定警察局出示适航证书或 NCT 证书。
- 在提出要求后十天内向警察出示有效机动车保险证书。警察在观察到或有合理理由相信未投保车辆曾在公共场所使用后一个月内，均可要求查看证书。
- 在任何合理时间出示车辆登记证书。
- 停车并允许警察检查车辆缺陷。
- 提供姓名和地址。
- 书写签名。
- 提供呼气或唾液样本。警察可随时要求道路旁样本，取样后必须留在现场等待检测。
- 接受“驾驶能力受损测试”。警察怀疑酒后或药后驾驶时可要求测试；开始前会询问是否有可能妨碍参与的残障或医疗状况。

警察可根据测试表现认定驾驶能力受损。拒绝按要求接受测试属于违法行为。

爱尔兰警察可以设置检查站抽取路旁呼气样本，无须事先认为驾驶人已饮用致醉物。拒绝提供样本属于刑事犯罪。

因酒驾相关违法行为或拒绝提供路旁呼气样本被捕后，须在警察局提供呼气、血液或尿液样本。

驾驶人涉及碰撞并在医院接受治疗时，警察也可要求允许护士或医生抽取血液或尿液。除医护人员本人基于医疗理由拒绝取样外，驾驶人拒绝许可属于违法行为。

无法自行配合时，例如失去知觉，警察可指示医生或护士抽血。之后拒绝允许检测该血样属于违法行为，除非有特殊或重大理由。医生认为采集血液或尿液会损害健康时，警察不会提出取样要求或发出指示。

## 其他道路执法

- 税务专员署人员，包括海关，也可以拦停并检查车辆。
- 税务官员或爱尔兰警察可以扣押车辆。
- 爱尔兰警察还可能与 RSA 运输官员共同拦车，检查行车记录仪和营运者执照。

## 紧急服务车辆接近时

看见紧急车辆时，应迅速、安全而谨慎地反应，让其通过。始终保持警觉，车内音量应能让你听见警报器。交叉路口尤其要留意四周，因为紧急车辆可能从后方或支路驶来。

停在车流中时与前车留足空间，以便需要时靠边让行。爱尔兰警察、消防员和救护人员执行救命任务，每一秒都很重要。

紧急车辆接近时：

1. 看见闪灯或听见警报器后，在能够安全进行时尽快让路。
2. 除非绝对必要，否则不要驶上路缘；即使必须这样做，也要确认那里没有行人。
3. 查看车内和两侧后视镜，判断紧急车辆速度，并留意行人、自行车、摩托车和其他道路使用者。发出准备靠边的信号。
4. 只在有足够空间让紧急车辆安全通过的地点靠边。
5. 留在原处直到车辆通过，并留意后方可能还有更多紧急车辆。
6. 进行适当观察，发出驶回信号，并在安全时逐渐汇入车流。

绝不要：

- 紧跟或超越正在使用闪灯或警报器的紧急车辆，除非获指示；
- 跟着紧急车辆冲过交通灯；
- 为让行而闯红灯或超速，除非爱尔兰警察或紧急服务人员指示；
- 突然制动或堵塞道路；
- 超越正在行驶并展示闪灯的紧急服务车辆。

### 行人、骑自行车者和摩托车骑手

应观察并聆听紧急车辆，判断其路线，确保自己不在行进路径上，并尽量靠道路正确一侧。

行人如附近有路堤或草质路肩，应在安全时使用。推婴儿车时可能无法这样做，应尽量靠边，并尝试预判紧急车辆路线，使驾驶人注意到自己。

带儿童时始终牵住他们的手。穿高可见度服装能让其他道路使用者看见你。行人在交通灯、行人灯、斑马线或 Pelican 过街处横过道路前，应等所有紧急车辆通过。
