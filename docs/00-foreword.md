# 前言

《道路规则》的宗旨是挽救生命，防止人们在道路上受伤。这些规则适用于所有道路使用者：驾驶人、行人、摩托车骑手、骑马者、骑自行车者以及负责看管动物的人。

本规则中凡提及“骑自行车者”，均包括脚踏自行车和电动自行车（e-bike）的骑行者，以及电动滑板车（e-scooter）使用者和 L1e-A 类电动轻便摩托车骑手。

这些规则要求我们对自己使用道路的方式承担个人责任，并由此成为更优秀、更安全、更具社会责任感的道路使用者。

《道路规则》采用易于阅读的版式，并以浅显的英语写成。阅读并了解这些规则是你自己的责任。你可以在本书术语表中查找我们使用的词语，这会对你有所帮助。本规则另有爱尔兰语版本。

这些规则符合并反映截至 2025 年 3 月 1 日的道路交通法规。如果我们已知法律将发生变化，便会在相关章节中加入说明。请务必查看道路安全管理局网站 [www.rsa.ie](https://www.rsa.ie) 的相关栏目以获取最新信息。

改变我们的行为，就能挽救生命并防止伤害。阅读并遵守这些规则，将有助于实现这一目标。

这是我们的责任。

这是我们的选择。
